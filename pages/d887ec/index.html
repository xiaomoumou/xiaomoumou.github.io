关于scss的用法

Sass 是一款强化 CSS 的辅助工具，它在 CSS 语法的基础上增加了变量 (variables)、嵌套 (nested rules)、混合 (mixins)、导入 (inline imports) 等高级功能。scss是sass的一种语法

# @for

@for 指令可以在限制的范围内重复输出格式

分为两种格式

  • @for $var from {start} through {end}

    条件范围包含 start 与 end 的值

  • @for $var from {start} to {end}

    只包含 start 的值不包含 end 的值

@for $i from 1 through 3 {
  .item-#{$i} { width: 2em * $i; }
}

# mixin参数

混合指令（Mixin）用于定义可重复使用的样式 @mixin用于定义(简写 = )，@include用于引入(简写 + )

// 可以添加默认值，如下
// @mixin sexy-border($color, $width: 1in) 
@mixin sexy-border($color, $width) {
  border: {
    color: $color;
    width: $width;
    style: dashed;
  }
}
p { @include sexy-border(blue, 1in); }
// 关键词参数
p { @include sexy-border($color:blue, $width:1in); }

# 函数指令

支持自定义函数，并能在任何属性值或 Sass script 中使用

$grid-width: 40px;
$gutter-width: 10px;

@function grid-width($n) {
  @return $n * $grid-width + ($n - 1) * $gutter-width;
}

#sidebar { width: grid-width(5); }
