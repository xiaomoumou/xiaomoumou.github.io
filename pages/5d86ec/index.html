    • 发现文档的配置方面很不齐全，
    • 计划去看vitepress版本v1.0的配置，应该是共通的吧？
  • 算法
    • 88. 合并两个有序数组 (opens new window)
      • 描述：合并两个非递减数组，nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目。
        • 注意：合并后数组不应由函数返回，而是存储在数组 nums1 中。
      • 例子：
        • 输入：nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
        • 输出：[1,2,2,3,5,6]
      • 解决思路：
        • 三指针：反向循环，
          • 指针一指向nums1末尾，从后往前，确认最终数字，
          • 指针二指向nums1中的循环
          • 指针三指向nums2中的循环
