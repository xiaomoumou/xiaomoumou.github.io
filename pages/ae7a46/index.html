# 一个axios上传的案例

# 1.【上传文件】input 标签之 file 类型

# 1.1 基础用法

input 标签 type 为 file 时，input 表现性状为上传文件样式。

关于file-input 的属性

  • accept：指定选择文件类型的范围。默认为所有文件类型
  • capture：当文件类型为图片或视频且在移动端时，此属性才有意义。
    • capture = ‘user’ 调用前置摄像头
    • capture = ‘environment’ 调用后置摄像头
  • multiple：一个 Boolean 值，如果存在，则表示用户可以选择多个文件
  • files：返回一个 FileList，列出每个所选文件对象。除非 multiple 指定了属性，否则此列表只有一个成员。主要用于 JS 操作。
<!-- html -->
<input type="file" multiple id="imgLocal" accept="image/*" />
<script>
// js
  let inp = document.querySelector('#imgLocal')
  inp.onchange = function(e) {
    let fileList = document.querySelector('#imgLocal').files
    console.log(fileList) //files
  }
</script>

# 2.【读取文件】FileReader 对象

# 2.1 基础用法

MDN FileReader (opens new window) FileReader 对象允许 Web 应用程序异步读取存储在用户计算机上的文件（或原始数据缓冲区）的内容，使用 File 或 Blob 对象指定要读取的文件或数据。

FileReader 对象常用事件和方法

  • FileReader.onload
    • 处理 load 事件。该事件在读取操作完成时触发
  • FileReader.readAsDataURL()
    • 开始读取指定的 Blob 中的内容。一旦完成，result 属性中将包含一个 data: URL 格式的字符串以表示所读取文件的内容。
// 继续使用上文的fileList
let file = fileList[0]
const fileReader = new FileReader()
// 读取图片
fileReader.readAsDataURL(file) 
// 监听读取完成
fileReader.addEventListener('load', function() {
  // 返回base64格式的图片
  let res = fileReader.result
})

<input type="file" multiple id="imgLocal" accept="image/*" />

  let inp = document.querySelector('#imgLocal')

      if (event.loaded && event.total) {
        // 计算完成百分比
        const percent = (event.loaded / event.total) * 100;
      }
    })
    console.log(size,url01,url02,percent)
  }
</script>

# 3.【组装表单对象】FormData 对象

FormData 对象的作用

  • 1.用一些键值对来模拟一系列表单控件：即把 form 中所有表单元素的 name 与 value 组装成一个 queryString
  • 2.异步上传二进制文件。
let formData = new FormData()
formData.append(name, value, filename)

2
3

关于 FormData()实例的参数

  • name:属性名
  • value:属性值，在我们这里则指 file 数据。
  • filename:当第二个参数为 file 或 blob 时，告诉服务器的文件名。Blob 对象的默认文件名是“blob”。

# 使用form表单上传

要添加属性 enctype="multipart/form-data"

form表单的enctype属性（method="post" 时才使用 enctype 属性）

  • 作用：告知服务器请求正文的MIME类型，规定在发送到服务器之前应该如何对表单数据进行编码。（请求消息头：Content-Type作用是一致的）
  • 可选值：
    • application/x-www-form-urlencoded(默认)：发送前对所有字符进行编码
    • multipart/form-data: 使用有文件上传控件的表单时，该值是必需的。
    • text/plain：将空格转换为 "+" 符号，但不编码特殊字符
