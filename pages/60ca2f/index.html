特拉福买家俱乐部

# 2018-03-05

# 遗弃

这里没有什么赵茹，这里只是来了一个小姑娘。

她说，她也是被遗弃的人。这里都是被遗弃的人，他们都没有过往。

仅此而已。

# 阳光

她说过：这里细小，简单，一天也只有一两个小时的时间阳光能够照进来。

就是这个时间。

她也说过：还是这里比较适合她。

# 2018-03-07

# 夫妻

“你会不会觉得作为丈夫的我，很没用？”

她看着他，好久一会儿，才忽然问道：“我饿肚子了吗？”

“应该没有吧？”

“我生小孩连一个病床也没有了吗？”

“有！卖血我也不能让你没床位啊！”

“你会让我委屈吗？”

“不会！”

“那谁说你没用了？”

她轻轻地抱了他一下，“我知道，你也会有发光发热的那一天。我呀，你只要饿不死我和接下来的孩子，就会安静地等着。”

他鼻子一酸。

那种梦想……在这个女人的面前，还能值几个钱？

# 2018-03-10

# 从容

这连自杀都显得从容的老人，此刻却在恐惧和拼搏之中，散着一种让人信服的力量。

# 2018-03-11

# 蹉跎

她也害怕一个人……

一个人吃饭，午饭，或者是晚饭。

中午的时候，两个年岁差了十几年的女人，在附近的餐馆里面蹉跎了一下时光，有一搭，没一搭。

说的是家长里短。

还真是蹉跎啊...

# 春天里的熊

“就像是春天里的熊。”他忽然说了一句话。

她露出了疑惑的目光。

“《挪威的森林》之中的一段形容。喜欢，就像是喜欢春天里的熊一样。”

“为什么是熊？”她下意识地问道。

他抬起头看着上空的多云，就像是要把它聚散不停的多云想象成小熊的模样，说：“春天的原野上，百花盛放，鸟语花香，你独自一人走在山坡上，对面一只毛茸茸的，黑白交错的小熊走来。它傻乎乎地坐在了你的们面前，朝着你眨眼，于是你把它抱着，然后从柔软的山坡上滚了下来。你不痛，因为小熊很柔软，因为青草和泥土很柔软。你的身上沾满了青草的草屑，还有野菊花的花瓣，此刻你不用想任何的事情，身心舒畅......不喜欢么？”

# 等三年

他俩就在那时候结交订了百年。

然后说过，哪个九十七岁死，哪个奈何桥上等三年，这样的话儿。

他就一直等着她了……

# 柏拉图

在一个浪漫的晚上，有着烛光和花瓣，在柔软的天鹅绒的床被之上。

刚刚接受完了亲朋戚友祝福正式成为了合法夫妻的洛邱和优夜二人，接下来本应该进行天地人伦的大事的时候。

可这两个家伙却衣服也不脱，只是牵着手，安静地躺在了床上，在玫瑰花瓣之间，感受着彼此之间的心跳。

柏拉图，是很唯美啦……

可是怎么造人？？！！

# 逃避现实

里面主人公的性格就很好啊……从开始就很好。

很羡慕啊……

这种从一开始到结尾，不管碰到什么痛苦、什么失败、什么挫折、什么失落、什么悲伤都能够说跨过去就跨过去的从一而终。

我们都把自己投射在那种理想的状态之中，借以逃避现实的吧？

# 2018-03-25

# 坚固的友谊

她并不怎么相信无私的友情，她只相信坚固的友谊。

而坚固的友谊，是需要经营出来的。

首先让自己不处于一直被接济的环境，才能够与友人有更好的发展空间，不让对方觉得自己是作为累赘，才能够在许多事情上，双方处于对等的倾谈状态。

听从友人的建议，会让对方产生一种愉快的感觉……

还有发问，询问对于对方来说如同常识一样，轻松就能够回答出来的问题……优越感。

# 记录

从什么时候开始，会有想要把一些瞬间拍下的想法呢……是觉得就算存在于记忆之中也不足够吗？

# 2018-03-26

# 愿你

愿你有好运气，如果没有，

愿你在不幸之中学会慈悲。

愿你被很多人爱，如果没有，

愿你在寂寞中学会宽容……

# 家人

“不让家人太过担心，不就是作为家人应该做的事情吗？”

# 2018-03-27

# 金盆洗手

所以你们就心安理得地当作自己的双手已经洗干净，在这里安稳的生活，儿孙满堂的儿孙满堂，父慈子孝的父慈子孝，对吗？

# 2018-04-28

# 鱼缸

“你觉得这池塘比鱼缸大，这些金鱼会更加自由一些对吗?”

“可是你想过了没有，在鱼缸当中，金鱼是安全的，甚至不用愁吃的问题，因为养它的人会按时地给它喂吃东西。可是在池塘就不一定了，它们不一定能够找到吃的，没有吃的东西就会饿死。”

“再来就说，池塘中除了金鱼之外，还有别的鱼，大鱼吃小鱼，很有可能，这些小金鱼肚子饿了，还没有找到吃之前，就已经被大鱼吃掉。”

# 2018-07-31

# 迎生送死

总觉得，医生当久了，看待人的死亡就会非常麻木，再惨烈的死亡，也无法激起悲伤。

我不喜欢这个样子，会为活着而喜悦，为死去而悲伤，这才是人。

# 2020-02-27

# 命运的如梦似幻

她是如此突然地出现在他的生命之中，如同命运的安排。

此时他不经心想，她会否也会如突然的到来般，也突然的消失。

让这一切看起来，只是一场梦幻？

# 2020-06-10

# 单身久了

如果你单身的时间长了，看什么都觉得清秀。

你会发现这个世界好看的女人实在太多了，怎么追也追不完，反而还会惹来一身的风流债。

所以，得出的结论是，当然还是自己一个比较舒服些。

# 2021-06-04

# 灵魂的颜色

“你说，我的灵魂是什么颜色的？”

“不是很亮，但也算不上暗淡。”洛老板一脸认真地道:“属于中规中矩的白色，不过很坚定，我还算满意这份坚定的。”

“为什么是白色?”

“因为黑色总是给人神秘一些的感觉。”
