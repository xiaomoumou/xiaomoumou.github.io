2021-11-22

我在风花雪月里等你

我们好像化身成为两条长了翅膀的鱼，洱海已经不是我们的束缚，而蔚蓝的大海才是我们真正的归处。

我们看遍了沿途的万千风景，从南极到北极，当我们回归到专属于我们的那片海域时，四季也轮换了一遍，而迎接我们的，恰好就是百花争艳的春天……

我们没有交流，可心灵已经相通……

# 2020-07-16

# 杀死孤独

在我的认知里，肉体的交融就像一剂良药，会杀死病毒一样的孤独

# 《再见二十世纪》

阿甘说生活是一块巧克力

我想：也许他是对的

一个女人说生活是孩子和房子

我想：也许她也是对的

上帝说生活是救赎和忏悔

我想：也许我是个罪人

我从五岁歌唱到现在已苍老，现在还是两手空空，像粒尘土

再见，二十世纪

再见，和我一样迷茫的人们……

# 离世

汪蕾的匆匆离世，让我对人生有了更多的思考。

我一遍遍问自己，我到底该怎么活着？

难道我这类人留在大城市真的只有死路一条吗？

其实并没有那么极端，但我肯定不会快乐，这里有无数痛苦的回忆。

我也没有太多希望能够赚到足够在这里立足的钱，

我好像一直活在对与错、是与非的漩涡里，疲于应付。

# 客栈

对这崭新的生活，好像也有了更多的体会：我就比以前在上海时要闲很多，不怕喝醉了会影响到第二天的工作状态；我也不必住在19楼的小房间里，对着窗外林立的高楼，感到无比压抑。

这里，让我看到了一个释放后的自己。

于是，心里更加期待早点将客栈做起来。

我想在未来养一条阿拉斯加，一起迎接从四面八方过来的客人。

我还要在院子里搭建一个可以看见洱海的小阳台，每天不忙的时候，都会坐在上面喝喝茶，看看平台外的风景。

这些，是我能代替汪蕾唯一能做的事。

# 腐烂或者超脱

铁男和马指导。一个浪荡，一个颓废，似乎一点也看不到积极向上的精神。这让我有些疑惑，他们到底是腐烂了，还是已经超出了琐碎生活，进入到了一个新的境界？

# 交待

我的精力忽然变得特别有限，我不想交际，不想关心别人，只想踏踏实实地在大理开一个能赚钱的客栈，在实现汪蕾梦想的同时，也让自己开始新的生活。

这种新生活，会让我有足够的底气向家人进行交待，而不是让他们看不到希望！

# 大理

大理好像就是这么一个地方，它总是会用它的闲适，把人变得爱胡思乱想，却将生活里必须要面对的实际问题给淡化了……

那么，这对于那些极端现实主义的人来说，可不就是一个地狱吗？

# 梦幻

路灯的尽头，我忽然想起：她是第二个坐在我摩托车后座上的女人，也是一场正在做着，却不愿意醒来的梦！

是的，如果有这样一个女人，愿意坐着你的摩托车，跟你穿梭在大理的风花雪月中，那这本身不就是一场梦幻吗？

# 2020-07-17

# 霓虹里的孤独

那天，我背着厚重的行囊，迎着闪动的霓虹走在拥挤的街头。

我遇到了很多与自己插肩而过的人，我们没有交流，他们之间也没有交流。

而那些从车灯里射出的光，就像是一道道虚拟的墙，将这个世界隔离成了一块又一块，但大家还是能互相看见，又互相冷漠着……

我就在想：这是一种多么悲壮的孤独啊！

可到底是一种什么样的力量，让我们走在同一条路上，却从来不是一条路上的人？

直到今天，我也没能完全弄明白！我只知道，我对上海这座城市是有恐惧的，就像潜在深水里，浮不出来！

# 海

那种虚幻的感觉，又在我的脑海里浮现出了出来……

我们好像化身成为两条长了翅膀的鱼，洱海已经不是我们的束缚，而蔚蓝的大海才是我们真正的归处。

我们看遍了沿途的万千风景，从南极到北极，当我们回归到专属于我们的那片海域时，四季也轮换了一遍，而迎接我们的，恰好就是百花争艳的春天……

我们没有交流，可心灵已经相通……

# 2020-07-19

# 我陪你

一句“我陪你”就像一剂良药，杀死了像病毒一样的孤独

# 2020-07-20

# 当面说

重要的事一定要当面说，有些话你看着她的表情，也许就不忍说出口了

# 热闹

我还是想在热闹的古城里找到一点过年的气氛，所以就特别想把自己淹没在人群里。

# 力不从心

杨思思咬住嘴唇，用一种能穿透的目光看着我，而这样一个她，会让我忘掉之前的她，总觉得她的身体里也不全是流于表面的快乐，她用来思考的大脑里，其实也有一样很深邃的东西。

而这种深邃，让我难以琢磨，也力不从心。

# 渴望

“你不是绝望，你只是累了。”

我夹着烟的手指，不自主的颤了一下，然后便千疮百孔；在这样一个不知道该何去何从的夜晚，我特别不希望有那么一个人能洞穿我的内心……因为，这会唤起我心中想获得依靠的渴望。

# 下雨

下雨会让大理这座城市变得非常冷，也会让你不自觉想掖起衣服，抵御这种冷清的感觉，然后你就好像与这个世界失联了。

即便衣服外面的世界，有纯真的孩子，美丽的少女，飘着香气的烤红薯，都会觉得与自己没什么关系。

这大概就是最高境界的孤独吧……

活在孤独的世界里，却比这个世界更孤独。

# 卑微

我渐渐将等待和卑微划上了等号，因为已经在长椅上坐了两个小时，吸了半包烟的自己，真的很卑微。

# 等待回复

在一条充满诚恳的信息没有得到回复之后，我已经不知道还能用什么样的文字去向她表达自己的想法和心情

# 热恋

这是我第一次对她说出“我爱你”这三个字，区别于“我喜欢你”，就像是一种特别正的仪式，我当然希望她能感受到我的真心。

打开手机屏幕，真的是叶芷回复的。

“我也爱你。”

这是一种甜到快要窒息的感觉，和大部分陷入热恋中的情侣一样，我们的快乐就来自于这样的话语中，会扫除疲惫和猜疑，让人更加憧憬以后的生活。

# 异地恋

异地恋最难熬的地方就是在这里，明明很想和她说一句话，却又不想打扰她休息，于是只能用某些他（她）的东西，来找到心理上的安慰，可这无疑是画饼充饥，因为人内心的欲望是无限大的……

所以我要的不仅仅是她的照片，我更希望她就睡在我的身边，哪怕各自玩手机，也比这种找寄托的感觉要好的多。

# 无爱亦无嗔

我突然对他现在的境界产生了好奇感……

他从此以后，真的不会快乐，也不会感到悲伤了吗？

# 取悦自己

你要是觉得这些东西能填补你内心的空虚，那你完全可以当成是一种取悦自己的方式……

# 等待

因为哪怕是等，也只有别人放弃我的份儿，我从来没有放弃过任何人。

# 2020-07-21

# 新生儿

红着眼眶的杨思思，又对我说道：“大理对于你，对于我，都是生命中的另外一个起点……我们就是在这个起点上认识的，我从来没有觉得这是缘分使然，在我眼里，我们就是两个想改头换面的人，为了同一个目的走到了一起……就像，每天一定会有很多新生儿在医院相遇一样……大理是我们的医院，我们在大理重生了！”

# 成熟

你只会去看外表，看言行举止……

但我觉得衡量一个人是不是成熟，得去看他（她）的内心，我的心里也藏着一座山和一片海，所以我才和你一样喜欢大理这个地方呀。

但我们也不完全一样，我心里的海很蓝，山很清秀，可你的山上没有树木，海也是灰色……

可你不能因为自己心里没有阳光，永远深沉，就觉得别人的乐观和快乐，是不成熟的呀！

# 爱的名义

我因此想过，我的人生中，是不是真的可以以爱的名义，将她永远留在自己身边？

当我这么想的时候，我莫名对自己没有信心。

# 家庭

就是这一刻，我的内心有了一阵想和她结婚的冲动……我想忘了关于大理的一切，然后在成都这个地方，组建一个家庭。

这大概就是一个男人的柔软了，会想成家，会想回报对自己好的女人，会想有个既有孩子也很温暖的家庭……而理想，或者说是梦想，会被放在一边。

# 城市的阴暗面

我就是这么不喜欢上海的……因为在它繁华的背后，有太多无能为力的人，他们都活在角落里，活在这座城市的阴暗面，就像跳蚤，难说尊严，难说为人。

# 爱人

必须有爱的人相伴，如果没有，也一定要把她幻想出来，这会成为生活的动力，然后把自己变成一个有向往，也有理想的人。

而大城市，只会把你变成机器。

# 女人

我一直认为：能将死缠烂打和欲擒故纵组合起来，用来对付女人的男人，都是花丛中的老手，因为他们都太了解女人的天性了

# 粥

在这一瞬间，我忽然特别想见到杨思思，然后将这一份精心做出来的粥送到她手上……

我的要求并不高，只希望她能当着我的面吃完，再给我一个笑脸，就足够了。

# 孩子

“米高，你说是男孩好玩，还是女孩好玩啊？”

“你生孩子就是为了玩？”

“当然了，如果生个孩子，还不能和孩子打成一片，那和生育机器有什么区别……反正我认为，大人和孩子之间的情感，很多都是在玩的过程中建立的，太严肃了，反而没人情味。”

我认真想了想，说道：“你要这么说的话，那我觉得女孩好玩……女孩要是长得好看，就特别像瓷娃娃，看着都开心！”

“不对，男孩好玩，越皮的越好玩……这样，我就可以每天打他一顿，先把他搓扁了，再把他给搓圆了！”

我先是愕然，然后又感叹道：“你不会是个虐待狂吧？”

“你听过一句话，叫父债子偿吗？我在他爸那儿受的委屈，就让他赔给我……”

“你这还真是对事不对人啊。”

“谁让他爸是个混蛋……”

# 过往

我甚至会设想：小时候的杨思思，沿着湖岸边，光脚奔跑的模样……

她就留着小马尾辫，沿着夕阳，那些残留的脚印，就是她最美好的时光，也是我心中所向往的。

可惜，时间上，我们却没有这样的交集。

# 杀死心爱之人

“我记得，王尔德说过这么一段话，大概是：人人都曾杀死心爱之人，你不妨听听每人的方式；有人使用冰冷的眼神，有人使用热烈的表白，胆小的使用轻轻一吻，无畏的挥出尖利的刀刃。有人毁所爱时还年少，有人毁所爱时已年老；有人用欲望之手扼杀，有人靠金钱之手诱杀；最慈悲的才使用利刀，为了让死者少受折磨……”

# 平和安定

我更希望，有那么一天，她会变成一只白鸽，停在我的心里，给我和平和安定的力量。

# 不幸

瞻前顾后，才是不幸的根源。

# 2020-07-22

# 体贴

“你说的对，我们之间最大的矛盾，就是都坚持着自己所认为对的事情，却从来没有静下来去考虑，对方到底需要什么！”

# 爱情

好似他的生活中，已经完全用不到爱情这个东西。

# 穷

穷这个东西，真的是一场大病，会让人自卑，让人失去信心……

# 爱

人们常说爱情是无价的，我是觉得，能这么说，本身就已经是一种衡量。

如果没有衡量，也不可能得出无价这个结论，而一旦有了衡量，必然不会纯粹……

所以，这个世界上从来都没有绝对纯粹的爱情，不愿意相信这个观点的，不是天真，就是没有真正经历过。

# 出家

“别人做和尚，是为了修行，他是真把和尚做成了一份职业……你相信，这一套房的钱，就是他做和尚这两年攒下来的吗？

# 家

我问自己：到底什么是家呢？非得有个房子吗？

未必，城市里那些大片、大片空着的房子是不能被称为家的，真正的家，一定会有两个相依为命的人。

# 残疾

我梦见自己的腿没有被治好，真的变成了一个残疾人；

我的人生也在随后跌进了无尽的深渊中，一开始，我很痛苦，可渐渐也接受了这种生活，我变得越来越普通，然后找了一个身体同样有缺陷的女人结了婚，我们之间没有爱情，但我们从来都不争吵，就这么互相尊重着过完了一生……

# 丢失

要是同时有两个女人这么对我，丢了谁，我都觉得是要了我的命……你倒好，最后两个都给弄丢了……

# 取舍

“我也不知道为什么，取舍着、取舍着，自己的身边就没人了，这回是真的没了！”

# 杨如画

她已经不再是那个，我随便一个抱怨，就能让她从上海飞到大理陪我吃早餐的女人。

# 矛盾

“嗯……我就是这个世界上最矛盾的人……我每天都想买一把刀捅死你……可是在去买刀的路上，想起来你爱吃苹果，就顺手买了一袋苹果，可最后刀却忘买了……但这说明不了什么，我还是想捅死你……就等我哪天记性变好了。”

“你现在也可以捅死我。”

“那不行，我给你买的苹果，你还没吃完呢。”

# 执迷不悟

“执迷不悔的另一面，叫执迷不悟……”
