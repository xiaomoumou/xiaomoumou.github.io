随想

# 2018-11-10

看完《间客》《雪中》的时候，就想着写点随笔，记录一下某刻的所思、所想。

想写些关于小说的思考，还有些对“看书”这件事的思考，大多时候由于想的太多，写的太少，所以需要写些随笔。

# 自私

看过的故事越来越多了，最喜欢那句“阳光底下，没有新鲜事”。

人间发生过的事情，过了的事，故去的事，故事。

看过的故事大多是小说里的，虽然现实比小说更加诡谲多变，上至家国下至柴米油盐的事更多。可我不喜欢，不愿听、不愿理，或许是因为眼不见就与我无关吧？自嘲“风声雨声读书声声声不入耳，家事国事天下事事事不关心”。

最自私的人，自私到只愿意把时间放在想自私的地方。

猫腻说“放眼四野直至星穹，追溯时光直至永恒，需要思考的事情太多，每个生命的时间太少，人间都不值得，情爱又算什么？”情爱不值得，人间不值得，只有自己所爱值得。

# 雅俗

无分对错、好坏、雅俗，只分有没有意思——或者说，“我”感不感兴趣。

一本世人所谓的名著对我而言没有任何意义，它和看过的任何一个故事都是一样的。只是加上了“名著”这个标签，才使它迥异于其他的故事——这里没有用“书”这个词，因为书只是故事的载体，而一个故事从不取决于它的载体。

名著的妙处——很多人说它好，所以没有人敢说它不好，所以容易让人产生阅读它的你很好的错觉。

# 动物

所有故事都是围绕着人来写——包括人与人、人与动物、人与自然。

从生物学上来说，人是动物的一种。所以很多故事里喜欢用野兽来形容人，当然，有些人的心性刚好像野兽，有些牲畜刚好长得像人。

那么本质上来说划分可以把故事重新划分——动物和动物、动物和自然。

曾经很认真的思考过，假如造就某个人的成长环境，换成另一个人，会不会造就出一个一样的人？

除开个人天生的性格，从环境培养出来的意识和眼界是一样的，那么性格也能由环境造就么？ok，实际上这只是不可能事件，人不能两次踏进同一条河流。

也曾思考过如果把动物换成人的位置呢？从小生活在狼群里的人类会向狼的方向发展，狼怎么往人的方向发展呢？答案是不可能，这是人类这个物种的优势。

除去动物与动物的部分，还有动物和自然的部分，动物和自然是怎么相处的？

物竞天择，适者生存——“适者”比“强者”重要，有时候“强”与“适”并不相等。提起人与自然的斗争，经常想起某个场景里，男人在飘摇的船上迎着狂风呐喊的画面。

不是很理解爬上了一座山就冠以征服的字眼——这是一件荒谬而可笑的事，实际上，山就在那里。蚂蚁攀上了人的躯体，就可以自称征服了人嘛？

# 百年

人类的伟大在于百万年间从猿猴进化成一种有思想的动物，而其他猿猴依然是猿猴。视角缩小到人这个层次呢？

百年从懵懂无知成长到依旧无知？

亦或是成为人间事事事知晓的智者？

哪怕是一百年的漫长时光，也是从这一刻开始的。

事事精通很难，准确的讲，每一件事精通都很难。事件是无限的，人生是有限的，需要思考的太多，所以只需知晓所需要知晓的。除去人间事，还有其他，关于生命，关于自由，以及那些美好的部分。

# 2019-04-09

# 漂泊

一个漂泊的人是不会羡慕漂泊的。

此外，今晚两个二十余岁的“男孩”在说着关于”除了年龄之外的成长”

在上海的妖风中走了二十分钟后,我们从原公司宿舍搬去了另一个宿舍。

任杨说：除了钱之外还有很多东西值得我们去追求。在满足个人所需之后，钱只是一个数字。他说身体的变化只是变老，那不是成长。

2021.11.22

任杨已经是两年前的同事了，自从2019.06.30离职之后，没有和那家公司的同事交流过任何一句话，往事过眼成云烟。

# 2020-02-27

# 所谓有趣

我发现所谓的“有趣”是阶段性的事情。

某个时间段你会特别想找人聊天，然后会看到很多信息，假装“有趣”，同时会认识一些人，但是这些关系通常并不长久。

“有趣”这层伪装褪下以后，所有人都很苍白乏味，失去新鲜感。

# 2020-04-13

# 心心念念的去大理

大理只是个符号，我们需要一块地方来享受生活，而不是忍受生活。

# 愤世嫉俗

整个社会的价值观都被裹挟。

车房，本质上就几块铁和几块砖，99%以上的人一辈子的生产换来这些东西，生产出来的资源全被1%的人吃了。

普通人一辈子的意义就是被人吸血敲骨吮髓？

# 2020-09-26

# 想

2019年的夏天，从第一份工作离职后我在家待了几个月。

经常在傍晚黄昏中，沿着河边的跑道奔跑，边跑边想着那些我看过的书，喜欢的故事，梦幻的世界，赖以期盼的美好。

幻想存在着那么一个个充满奇幻怪诞迥异无趣现实的世界。

沮丧的是，幻想永远只能存在幻想中真是一件令人难过的事。

偶尔也会思考真实，怎么才算是一个人的成长，从小男孩到青年人，再到中年，除了身躯变老之外，还有什么能证明我真正的活过？

# 2020-12-20

# 骗自己

现在是凌晨5:30，窗外传来扫帚一遍遍划过水泥路面的声音，环卫工人开始了他们的一天。

我蜷缩在被窝，眼圈浓重的脸上映射着手机屏幕的光线，身边在播放着游戏直播，如同往常的每一个熬过的夜。

生活啊，浓浓的颓废气息。

我是个没有耐心的人，也不够深刻，没有特别铭记在心中的信念，也没有使命感般必须完成的任务。

我唯一的使命就是好好活着，享乐，不枉来人间一趟。

从明天起，要热爱游戏，热爱生活。

对决定喜爱的游戏充满耐心，多看剧情线，收获浮于表面下的故事。对身边关系良好的人保持更好的心态，要学会幽默风趣，希望能做到吧。

# 家庭父母孩子

纯理性来讲，你的生命之所以存在只是因为你的父母需要一个孩子。

所以他们生下一个孩子并抚养长大，家庭以及周围的环境形成了现在的你。

生命的意义这个词毫无意义，重点是你经历的事对你内心的影响（情绪，体验，感受之类的），从生到死，只有这一件事最重要。
