若星汉天空

# 2018-01-22

# 坚信

“你所相信的，你会一直这样坚持吗？”

“当然。”

“不，这世上一切都会变的。你也一样。”

“为什么你对未来这么悲观？”

“因为……未来本来就是希望消亡的过程，一切都将展现在你的眼前，再不会有悬念，不会有盼望，你以为你所能追寻到的一切，最后都消失在遥边的天际。”

“你阻止不了我，也许未来真得和你说得一样糟，但对我来说，没有比呆在这里更糟......”

# 很快

“那我们怎么办呢？等我们赚到钱，那女孩的孩子都该长大娶妻了吧。”

“不要这样想…我们很快就会有钱了。”

“昨天你也是这样说的……”

# 2018-01-31

# 魔都

魔都是个奇怪的地方，没有一座建筑是风格相同的，但是却涂成共同的黑色。

你会觉得走进了一块黑板，却无数不现出诡异的线条，那些线条可以无限的远近组合，所以人们每天早上醒来都发现自己在一个新的地方。

还是黑暗使人安心，光线中总有无数注视你的眼睛，而这里，只有永恒的水流声，你静听着，一直到时间的苍老。
