使用vue时遇到的问题

# watch

使用watch遇到的问题

# watch当中的this指向

问题描述：在watch函数里使用this获取data里面的值时，this指向undefined

export default{
  ...
  data:{
    name:'张三',
    age:18
  },
  watch:{
    age:(val,oldVal)=>{
      console.log(val, oldVal)
      console.log(this.name)   // this 为undefined
    }
  }
  ...
}

原因：使用普通函数时this才是vueComponent对象。 vue当中watch使用this的注意项

// vue代码编译前
var app = new Vue({
   el: '#app',
   data: {
    a: 1
   },
 
   watch: {
    a:()=> {
     console.log(this)
    }
   },
 
   methods: {
    onClick() {
     this.a++;
    }
   } 
  })
