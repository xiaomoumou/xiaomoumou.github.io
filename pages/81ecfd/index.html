git

代码管理工具，方便随时下载自己的代码

# 安装

官网 (opens new window)

安装完毕需要配置用户信息

git config --global user.name "Your Name"
git config --global user.email "email@example.com"
1
2

# 常用

# 初始化

在本地创建的文件夹执行

git init
1

# 关联

在本地创建仓库之后，需要在远程也创建仓库，然后本地仓库关联远程仓库

git remoto add origin (远程仓库的地址)
1

如果是第一次关联，需要进行ssh验证

# 提交

# 未跟踪的代码提交到缓存区
git add .  

# 本次提交的信息，方便记忆
git commit -m "初始化"

# 推送到远程仓库
git push -u origin master
