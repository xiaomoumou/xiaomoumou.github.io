generator

# 简介

generator（生成器）是ES6标准引入的新的数据类型。

# 应用场景

# 使用

function* foo(){
  yieId x+1;
  yieId x+2;
  return x+3
}

# 和函数的区别

  • 返回值 一个generator看上去像一个函数，但可以返回多次。
  • 定义方式 generator由function*定义（注意多出的*号），并且，除了return语句，还可以用yield返回多次。
  • 执行 foo()创建了一个对象，但是没有执行，执行的话，可以通过该对象的next()方法，或者使用for ... of循环迭代generator对象
