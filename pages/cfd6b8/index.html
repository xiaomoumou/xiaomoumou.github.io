# 2017-02-20

# 食物

让我们先撇开一些情感立场，从一个中立，并且客观，理智的角度来看待人类。

人类对待这个世界上的其他生物是什么态度呢？

最基本的属性是什么呢？

毫无疑问，是……

食物！

人类是杂食动物。

小麦也好，水稻也好……这些都是植物。

鸡鸭牛羊，家禽牲畜……这些都是动物。

而这些植物和动物，对人类而言，基本属性是什么呢？

毫无疑问，首先是食物！

也就是说，当人类和这个世界上的其他动物，以及植物的关系，首先的基本属性是食用和被食用的关系。

至于植物还有其他用途，比如说，可以当做木材，被人类用来砍伐建造工具，房屋。

有的人还会养植物，当做观赏。

还有的人，会养动物当宠物，爱护呵护，视为家人。

有养猫狗的，有养鸡鸟的，还有养乌龟的，养蛇……五花八门，各种各样。

但是首先，首当其冲的，人类和这个世界的其他生物的关系，排在第一位的，最原始，最基本的“初始设定”，说起来依然很残酷。是……食用和被食用的关系！

哪怕是一个素食主义者，一个狂热的爱护动物的虔诚信仰者。

一个素食主义者虽然可能不吃其他的动物，但是他（她）依然还是需要食用植物，小麦，水道等等。

一个狂热的爱狗之人，也许依然会食用牛羊猪鸡鸭等等……

也就是说，别管人类对待其他生物的态度会多出多少这种“额外属性”，宠物也好，爱护也好，呵护也好，保护也好……

但是，归根到底，人类对待这个世界其他生物的最原始最初始的设定，排在首位的基本属性，依然是……

食用！

这是事实，很丑陋很残酷的事实。

丑陋的事实。

这是生物进化的基本特性和原始天性决定的。
