死人经

我常常在想，这本书之所以让我印象深刻，是真的因为故事很精彩，还是因为我一直对里面刻画的人念念不忘？

印象最深的一个观点：有利用价值才不会被放弃，父子、亲友、恋人，亦是如此。

生者悲苦，死人平安

# 2018-09-15

# 言语

语言是座迷宫,为了试探对方隐藏自己。

说话者还要给这座迷宫添上更多的烟雾,得是熟悉规则的人,才能顺利趟过迷宫。

# 相信

“每个人一生中都有一次机会相信另一个人，我把机会给你。”

上官如实现了自己十二岁的诺言，即使经历一连串背叛与仇恨，即使早已物是人非，她仍然相信。

顾慎为感到难接受，他早已将世间一切规则抛在脑后，表面上的遵守不过是为了笼络人心，因此，他厌恶心绪的波动，将其视为软弱的表现。

# 依恋

这是一个教训，不要让任何一个女人和男人长时间待在一起，不管年龄与地位差别有多悬殊，总会产生依恋，当你不够坚定的时候。就会将依恋误认为爱情。

# 皮囊

我只是希望帮助一位年轻的英雄认清自己，请相信一个将死之人的经验。

你现在看得比生命还重要的事物，很快就会变得跟灰尘一样轻贱，不要被女人迷惑，你所爱上的只是一副皮囊，而这世上还有数不清的美丽皮囊等着你去发掘。

# 2018-09-17

# 蠢丫头

跟这个女人一比，他之前留宿的妓女简直就是乡下蠢丫头，只会虚假地媚笑，毫无技巧。

# 言语

语言多简陋啊，他想，越是含糊的东西越容易描述，精细到一定程度就会超出语言的樊笼，对之描述得越多，离真相反而越远。

# 付出与回报的偏差

“有人跟我说过，奴仆服侍主人其实是有代价的，他将自己的每一次卑躲屈膝都看作付出，日积月累，等着相应的回报，可主人却常常将之视为理所应当，这是许多背叛的萌芽。”

付出与回报，中间总是存在着偏差，不只主人与奴仆这样，一切人与人的关系都是这样，我要提醒你，

# 百态

他已经不习惯以好与坏、善与恶来判断一件事。

在见识过数不尽的阴谋与背叛之后，他坚信，好坏善恶、是非曲直皆属虚幻。

唯一真实的是生存，由生存引申出利益，由利益演绎出人生百态。

# 2018-09-25

# 价值

她坚信一件事：只要自己还有利用价值，他是不会轻易放手的。

# 窥视

顾慎为转向莫林，神色仍然一如既往的阴郁，好像总在想着自己的心事，要不就是在躲在面具后面窥视他人的心事，这不是王者该有的气质，莫林对此稍感遗憾。

# 庸脂俗粉

“庸脂俗粉也有美的一面，就是不要太多太浓，很少有女人知道适可而止，男人也不知道，大家只不过像猪狗一样打滚罢了。”

# 十年

“我等你十年。”说这话的人不知道十年是多久，更不必说百年千年。

“我愿为你赴汤蹈火万死不辞。”发这种誓言的人，身上连道伤疤都没有，以为死亡是一件遥远而且容易的小事。

顾慎为是说惯谎言的人，所以最清楚不过承诺的脆弱，大多数时候，人们并非刻意说谎，他们只是对自己说出的话没有真正深入的了解。

当他们实现不了承诺的时候，自会找出种种借口为自己辩解。

# 2018-09-26

当他用完美与成熟这两个词汇时，心中其实只有极为模糊的印象，就像是人们谈起某件自己从未见过的东西。

# 2018-11-23

# 泪水

泪水有什么用？当他还是小孩子的时候，这一招或许能带来安慰与帮助，现在却只是意味着软弱与耻辱。

# 2019-07-04

# 另一个世界

离别时顾慎为脑海里两人在莹莹星空下去往灯火辉煌的壁玉城。

那时候上官如梦想周游天下，欢奴渴望复仇，两人在夜色中形如一体。

如今顾慎为大仇得报，上官如也即将云游四海，他们却要分别了。

而壁玉城的灯火、背着少女的少年，正如顾慎为所想——那已经是另一个人、另一个世界的故事了。
