我常常在想，这本书之所以让我印象深刻，是真的因为故事很精彩，还是因为我一直对里面刻画的人念念不忘？

印象最深的一个观点：有利用价值才不会被放弃，父子、亲友、恋人，亦是如此。

# 2018-09-03

# 生者悲苦

魂升九天，魄落九渊，生者悲苦，死人平安。

# 言语

语言是个奇妙的东西

埋藏在心里的时候像一粒不知名的种子，可以随着主人的心意长出截然不同的结果来

一旦说出口，这些转瞬即逝的词汇立刻具有了固定的形态，对说话者具有了某种程度的束缚作用。

# 活着

有时候，死亡是一种福气，活下来的人却要忍受更大的痛苦。

# 找出真相与解决问题

“找出真相”与“解决问题”是截然不同的两件事。

打个比喻，有人丢了一百两银子。

追查真相的人想知道钱到底存不存在、有没有丢、谁是窃贼。

解决问题的人则想变出一百两让大家都满意，至于钱从何而来由谁提供，都是无关紧要的小事。

# 历史

历史告诉我们很多教训，其中一条就是没有人会吸取教训。

# 斩断关系

想要“安全杀人”，就必须在杀人之前斩断目标的“关系”。

这句话反过来的意思就是，如果不能斩断目标的“关系”，那就斩断自己的“嫌疑”，不要让任何人怀疑到自己头上。

# 是非曲直皆属虚幻

他已经不习惯以好与坏、善与恶来判断一件事。

在见识过数不尽的阴谋与背叛之后，他坚信，好坏善恶、是非曲直皆属虚幻。

人世中唯一真实的是生存，由生存引申出利益，由利益演绎出人生百态。

# 权力

权力是天生的，但它不是送给你的，而是你主动拿来的。

# 战胜情绪

在乎也好，不在乎也罢，都不要欺骗自己，战胜情绪跟战胜敌人一样，首先你得承认它的存在。

# 提也无用

那些最深切的痛苦提也无用，它不会在倾诉中减弱或是得到宣泄，反而更加显示幸存者的无力与软弱。

# 记忆

记忆像一条讨好主人的狗，只要你做出扔的动作，它总会给你叼回来点什么。

# 死亡

死亡遥不可及时的模样与近在眼前时截然不同。

远望如山，不露狰狞，巍峨壮丽，令人心动，甚至能激起诗情画意来，近看如崖，深不见底，无情的巨口让最勇敢的人也觉得四肢发软。

# 谎言

顾慎为是说惯谎言的人，所以最清楚不过承诺的脆弱。

大多数时候，人们并非刻意说谎，他们只是对自己说出的话没有真正深入的了解。

# 依恋

这是一个教训，不要让任何一个女人和男人长时间待在一起。

管年龄与地位差别有多悬殊，总会产生依恋，当你不够坚定的时候。

就会将依恋误认为爱情。

# 两种背叛

背叛有两种。

一种源于仇恨，永远无法化解。

一种源于自利，严格来说，那只是一种摇摆，是正常人在危急时刻的本能反应，只要形势转变，他们的态度也会改变。

如果非要把事情查个清清楚楚，，第二种背叛就有可能转为第一种，再也不能挽回。

# 不忘深仇之人必有大爱

不忘深仇之人必有大爱，由爱生恨，即是以善养恶。

世人为善而做恶，没有人为恶而做恶。

人人皆有一颗善心，偶尔为魔念所执，刻意曲解为善之道，才做下那些恶事，所以佛法有云“放下屠刀立地成佛”

# 名

名是虚幻的，不会给拥有者增添丁点力量。

名也是真实的，善于利用它的人，能将手中的力量放大千百倍。

# 解决大祸

解决大祸的手段就是惹一场更大的祸事。

# 醉

我醉的恰到好处，听的懂你的话，却不会感到悲伤。

# 语言是座迷宫

语言是座迷宫，为了试探对方隐藏自己。

说话者还要给这座迷宫添上更多的迷雾，得是熟悉规则的人，才能顺利趟过迷宫。

# 做好事

做好事跟做生意一样，你得有能力保证连本带利收回，当心血本无归。

# 兴盛与衰亡

兴盛与衰亡，即使通晓其中的一切规律，还是没办法加以掌控。

# 大人物的仰慕者们

人们就只看到站在高处的你，谁会在乎你用了什么手段、脚下踩了多少具背后有伤的尸体。

# 无耻的世界

这是一个无耻的世界，想在这个世界生存，你得无耻，想出人头地，你得加倍无耻，想战无不胜，你得抢在对手之前无耻。

# 主仆

主人往往将仆人的卑躬屈膝视为理所当然。

仆人却不这样认为，他们将自己的每一次曲意奉承和忍辱受过都当成做生意的本钱，一笔笔记下，等着适当的回报，如果没有回报或者回报不如意，背叛的种子就会生根发芽。

# 忠诚与背叛

主人都是愚蠢的，他们心安理得地享受一切，从没想过一切都是有代价的。

所以，你永远都能找到理想的背叛者，就是那些最忠诚、最诌媚、自以为付出最多的人。

# 领袖

对一群人的领袖来说，前进还是后退，永远是艰难的选择。

掌握的信息越多，选择越难，向前走直闯陷阱，向后走没准也在敌人的计划之内。

# 大赢家

选择即是赌博，无数有领袖潜质的人一遍遍掷下骰子，有人成功有人失败，运气最好的人成为最后的大赢家，被当做“英雄”。

# 潜力

有一些能力隐藏颇深，不仅外人难以发现，就是持有者本人也常常浑然不觉。

只有进入到某种特殊的境地，潜伏已久的能力才会在瞬间迸发。

于是。

沉默寡言者滔滔不绝天花乱坠。

懦弱无能者仗义执言拔刀相助。

在突发而至的危机关头，挺身而出的领导者可能是平时的碌碌无为之辈。

扛住木梁救出幼童的却是手无缚鸡之力的女子。

# 友情

友情是个奇妙的东西，有时比爱情还复杂。

对你最好、帮助最大的人未必能成为你最喜欢的人，反而是某个突然推心置腹过后又忘得干干净净的坏家伙，与你成为莫逆之交。

# 懦弱的面具

愚蠢与固执有着一副同样的面孔，而又常常带着懦弱的面具。

# 苦难

# 欢奴

顾慎为的故事终于落下了帷幕，这个故事描绘了一个非典型江湖。

大多数人印象中的江湖儿女是怎么样的？

快意恩仇如“事了拂衣去深藏功与名”？

或肆意洒脱纵马笙歌饮酒？

亦或是金庸老爷子笔下的“侠之大者为国为民”？

如果说大多故事描写的是江湖的光明面，那么《死人经》则在不加掩饰地述说着它的阴暗。

书里没有太多大义凛然、豪情万丈，只有赤裸裸的弱肉强食和淋漓鲜血。

不详意味的书名与书里多次出现的“生者悲苦，死人平安”，这八个字脱离了这个故事便真的毫无价值么？

故事里还有太多值得思考的问题。

找出答案与解决问题的区别？

忠诚与背叛？

一生只会存在一次的绝对信任。

先谈谈名字这部分。

父亲顾仑给他取名“顾慎为”，取意于“行善不以为名而名从之，名不与利期而利归之，利不与争期而争及之，故君子必慎为善。”

到后来经历灭门惨祸，沦落金鹏堡之后的名字——欢奴。

“欢奴”这个名字取自《千字文》里“散虑逍遥，欣奏累遣，戚谢欢招”的“欢”字，而“奴”则是奴才的意思，。

如果小说写完第一卷就结束，那它便是一本神作。

第一卷描述的是主人公由顾慎为到欢奴的故事，从幸福家庭中受亲人溺爱的小少爷，蜕变称为杀手组织金鹏堡中满身刀创剑痕的少年杀手，当中的心酸苦涩不足为外人道也。

作为旁观者的我，时常在想，人的赤子心性是在什么时候被杀死？

在死去的躯壳里又是什么时候孕育出了世俗的灵魂？

世俗杀人，如钝刀割肉，一割数十年，最终所有人都变得面目全非了。

他懂的第一个道理是：泪水是无用的，当他还是个小孩子的，这一招或许会带来帮助，现在却只会带来屈辱。

在金鹏堡的情节，描述了江湖的残酷性，冷漠，残暴，不折手段，大侠之所以人人向往，正是因为那是成人的童话，可遇不可求。
