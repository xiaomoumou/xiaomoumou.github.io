                # 行内式

                • 描述

                  []里写链接文字，()里写链接地址, ()中的“”中可以为链接指定title属性

                • 示例

                  这是超链接 (opens new window) 这是超链接 (opens new window)

                • 代码

                  这是[超链接](https://www.baidu.com)
这是[超链接](https://www.baidu.com "超链接")

                # 参考式

                • 描述

                  文中的写法 [链接文字][链接标记]，在文本的任意位置添加[链接标记]:链接地址 “链接标题”

                • 示例

                  我经常去的几个网站Google (opens new window)、Leanote (opens new window)以及百度 (opens new window)

                  你也可以写成[链接文字][],[链接文字]：链接地址的形式

                  Leanote 笔记 (opens new window)是一个不错的网站 (opens new window)。

                • 代码

                  我经常去的几个网站[Google][1]、[Leanote][2]以及[百度][3]
[Leanote 笔记][2]是一个不错的[网站][]。

[1]:https://www.google.com "Google"
[2]:https://www.leanote.com "Leanote"
[3]:https://www.baidu.com "百度"
[网站]:https://www.bilibili.com

                # 自动链接

                • 描述

                  只要是用<>包起来， Markdown 就会自动把它转成链接。

                • 示例

                  http://example.com/ (opens new window) address@example.com

                • 代码

                  <http://example.com/>
<address@example.com>

                # 锚点

                • 描述

                  在你准备跳转到的指定标题后插入锚点{#标记},在文档的其它地方写上连接到锚点的链接。

                  只支持在标题后插入锚点

                • 示例

                  Markdown书写风格{#markdown书写风格}

                  跳转到Markdown书写风格

                • 代码

                Markdown书写风格 {#markdown书写风格}
跳转到[Markdown书写风格](#markdown书写风格)

                # 注脚

                • 描述

                  在需要添加注脚的文字后加上脚注名字[^注脚名字],称为加注。 然后在文本的任意位置(一般在最后)添加脚注，脚注前必须有对应的脚注名字。

                • 示例

                  使用 Markdown1 (opens new window)可以效率的书写文档, 直接转换成HTML[^超链接], 你可以使用 Leanote引用 编辑器进行书写。

                  [^超链接]:HyperText Markup Language 超文本标记语言

                • 代码

                  使用 Markdown[^1]可以效率的书写文档, 直接转换成HTML[^2], 你可以使用 Leanote[^Le] 编辑器进行书写。

[^1]:Markdown是一种纯文本标记语言

[^2]:HyperText Markup Language 超文本标记语言

[^Le]:#表格

                # 引用

                • 描述

                  引用需要在被引用的文本前加上>符号，可嵌套使用>。

                • 示例

                  这是一个有两段文字的引用,

                  无意义的占行文字1.

                • 代码

                  > 这是一个有两段文字的引用,
  >> 无意义的占行文字1.
